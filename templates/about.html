About US

   
  This is our final project for DSCI 551.
 We used public api https://aqs.epa.gov/aqsweb/documents/data_api.html to gather historical data and airvisual https://www.iqair.com/air-pollution-data-api for real time data.  We used bootstrap for front-end design.
  Data is stored in MySQL database.
 We used spark to calculate average air quality for the past years.
